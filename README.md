# Secure Web Server with SSL & Reverse Proxy (Linux + DevOps)

This project demonstrates how to deploy a secure web server using:
- Nginx reverse proxy
- SSL/HTTPS using Let's Encrypt
- Linux server configuration
- CI/CD pipeline using GitHub Actions
- Optional Docker & Ansible automation

## Features
✔ Reverse proxy setup  
✔ Secure HTTPS  
✔ Backend forwarding  
✔ Automated deployment
